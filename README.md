# Caderno

Caderno é um editor de texto simples desenvolvido em Python utilizando a biblioteca Tkinter. Ele permite criar, abrir, editar, e salvar arquivos de texto com facilidade, além de oferecer funcionalidades de formatação como alinhamento de texto e controle de quebra de linha.

## Funcionalidades

- **Novo Arquivo:** Crie um novo arquivo de texto, limpando o conteúdo atual.
- **Abrir Arquivo:** Abra um arquivo de texto existente e visualize seu conteúdo na área de texto.
- **Salvar Arquivo:** Salve o conteúdo atual da área de texto em um arquivo, especificando o nome e o local.
- **Salvar Como:** Salve o conteúdo em um novo arquivo com um nome e local diferentes.
- **Sair:** Encerre a aplicação.
- **Quebra de Linha:** Ative ou desative a quebra automática de linhas na área de texto.
- **Alinhamento:** Alinhe o texto à esquerda, ao centro ou à direita.

## Instalação

1. Clone o repositório do GitHub:
   ```bash
   git clone https://github.com/sidneylcarneiro/caderno.git
   ```

2. Navegue até o diretório do projeto:
   ```bash
   cd caderno
   ```

3. Execute o programa:
   ```bash
   python caderno.py
   ```

## Como Usar

1. **Criar ou abrir um arquivo:** Utilize o menu "File" para criar um novo arquivo ou abrir um arquivo existente.
2. **Editar texto:** Digite o texto diretamente na área de texto.
3. **Salvar o arquivo:** Use as opções "Save" ou "Save As..." no menu "File".
4. **Formatar o texto:** No menu "Format", selecione as opções de alinhamento e quebra de linha conforme necessário.

## Requisitos

- Python 3.x
- Tkinter (geralmente incluído na instalação padrão do Python)
